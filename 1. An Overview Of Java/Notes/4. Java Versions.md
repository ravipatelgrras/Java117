# Java Versions :-
## 1. The Beginning (1991 – 1995)

- **1991**: Java was initiated by **James Gosling**, Mike Sheridan, and Patrick Naughton at Sun Microsystems.
    - Originally called **"Oak"** (named after an oak tree outside Gosling’s office).
    - Part of the **Green Project** aimed at developing software for embedded systems (like set-top boxes).

- **1995**:
    - Oak renamed to **Java** (Oak was already trademarked).
    - Officially launched as a **web programming language**.
    - Introduced the famous slogan:
      > **"Write Once, Run Anywhere (WORA)"**
    - Platform independence achieved via **Java Virtual Machine (JVM)**.

---

## 2. Early Java Versions

### Java 1.0 (1996)
- First official release.
- Features:
    - Basic language + JVM
    - Core libraries
    - **Applets** (Java programs running in browsers)

---

### Java 1.1 (1997)
- Added:
    - Inner Classes
    - JavaBeans
    - RMI (Remote Method Invocation)
- Improved:
    - Event delegation model for GUI

---

## 3. Java 2 Era (1998 – 2000)

### Java 2 Platform
- Introduced three editions:
    - **J2SE** (Standard Edition)
    - **J2EE** (Enterprise Edition)
    - **J2ME** (Micro Edition)

- Key Features:
    - Swing (advanced GUI)
    - Collections Framework
    - Strong security model

---

## 4. Modern Java Evolution

### Java 5 (2004) ⭐ Major Milestone
- Renamed to **Java SE 5.0**
- Introduced:
    - Generics
    - Annotations
    - Enhanced for-loop (for-each)
    - Varargs
    - Enum
    - Autoboxing / Unboxing

---

### Java 6 (2006)
- Focus on:
    - Performance improvements
    - Web services
- Enhancements:
    - JDBC 4.0
    - Scripting support
    - GUI improvements

---

### Java 7 (2011)
- Introduced:
    - try-with-resources
    - Diamond operator (`<>`)
    - NIO.2 (new file system API)
    - Switch with String

---

### Java 8 (2014) (`Revolutionary Release`)
- Introduced **Functional Programming**
- Features:
    - Lambda Expressions
    - Streams API
    - Optional Class
    - New Date & Time API (`java.time`)
    - Default methods in interfaces

---

## 5. Oracle Era & Rapid Releases (2017 – Present)

> Java shifted to a **6-month release cycle**

---

### Java 9 (2017)
- Introduced:
    - **Module System (Project Jigsaw)**
    - JShell (interactive REPL)
    - Improved Stream API

---

### Java 10 (2018)
- Introduced:
    - `var` keyword (local variable type inference)

---

### Java 11 (2018) ⭐ LTS
- Long-Term Support (LTS)
- Features:
    - New HTTP Client API
    - Removed Java EE & CORBA modules

---

### Java 12 (2019)
- Switch expressions (preview)
- Performance improvements

---

### Java 13 (2019)
- Text Blocks (preview)
- Switch enhancements

---

### Java 14 (2020)
- Features:
    - Records (preview)
    - Pattern Matching for `instanceof` (preview)

---

### Java 15 (2020)
- Text Blocks finalized
- Hidden Classes introduced

---

### Java 16 (2021)
- Records finalized
- Pattern matching improvements

---

### Java 17 (2021) ⭐ LTS
- Sealed Classes
- Pattern Matching enhancements
- Strong encapsulation

---

### Java 18 (2022)
- Simple Web Server
- UTF-8 as default charset

---

### Java 19 (2022)
- Virtual Threads (preview)
- Structured Concurrency (preview)

---

### Java 20 (2023)
- Continued improvements in Virtual Threads
- Scoped values (preview)

---

### Java 21 (2023) ⭐ LTS
- Virtual Threads (finalized)
- Pattern Matching for switch
- Record patterns

---

Java has evolved from a small embedded system language to one of the most powerful and widely used programming languages in the world.

> From **Applets → Enterprise → Cloud → Microservices → Modern Java (Functional + Reactive)**
