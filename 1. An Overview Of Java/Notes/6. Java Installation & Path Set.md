# ☕ Java Installation & Path Setup

---

## Introduction

Before writing Java programs, we need to:

1. Install Java (JDK)
2. Set environment variables (PATH)

---

## 1. Download JDK


::contentReference[oaicite:0]{index=0}


### Steps:

1. Go to official website:
    - https://www.oracle.com/java/technologies/downloads/

2. Choose:
    - JDK version (latest recommended)

3. Download for your OS:
    - Windows / Linux / Mac

---

## 2. Install JDK

### Windows Installation Steps:

1. Open downloaded `.exe` file
2. Click **Next → Next → Install**
3. Finish installation

📁 Default Path:
C:\Program Files\Java\jdk-XX

---

## 3. Verify Installation

Open Command Prompt and type:
java -version
javac -version

 If installed correctly, version will be displayed

---

## 4. Environment Variable (PATH Setup)


::contentReference[oaicite:1]{index=1}


### Why PATH is Required?

- To run Java commands from anywhere in CMD

---

## 5. Steps to Set PATH (Windows)

### Step 1: Open Environment Variables

- This PC → Right Click → Properties
- Advanced System Settings
- Click **Environment Variables**

---

### Step 2: Set JAVA_HOME

Click **New (System Variable)**
Variable Name: JAVA_HOME
Variable Value: C:\Program Files\Java\jdk-XX

---

### Step 3: Update PATH

- Select **Path** → Click **Edit**
- Add new entry:
  %JAVA_HOME%\bin

---

## 6. Verify PATH Setup

Run again:
java -version
javac -version

 Should work from any directory

---

---

## 9. Common Errors & Fixes

| Error | Reason | Solution |
|------|--------|---------|
| java not recognized | PATH not set | Set PATH correctly |
| javac not found | JDK not installed | Install JDK |
| Wrong version | Multiple Java versions | Update JAVA_HOME |

---

## Key Points

- Install **JDK** (not only JRE)
- Set **JAVA_HOME**
- Add **bin folder to PATH**
- Verify using `java -version`

---

## Conclusion

After installation and PATH setup:

- You can compile and run Java programs
- System is ready for development

> Installation is the first step to becoming a Java developer 🚀
