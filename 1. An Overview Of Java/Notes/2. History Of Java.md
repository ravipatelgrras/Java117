# ☕ History of Java

---

## Introduction

Java is a **high-level, object-oriented, platform-independent programming language** that was created to solve the problem of running the same program on different devices without modification.

---

## 1. Origin of Java (1991)

- Java began in **1991** at **Sun Microsystems**
- Developed by:
    - James Gosling
    - Mike Sheridan
    - Patrick Naughton

- It was part of a secret initiative called the:
  Green Project

### Goal of the Project

- Build software for **consumer electronic devices**
- Example:
    - Set-top boxes
    - Smart TVs
- Needed a language that was:
    - Platform-independent
    - Reliable
    - Efficient

---

## 2. Birth of Oak

- The first name of Java was:
  Oak

### Why "Oak"?

- Named after an **oak tree outside James Gosling's office**

### Features of Oak

- Platform-independent
- Object-oriented
- Designed for embedded systems

---

## 3. Problems with Oak

- Oak was **technically strong**, but:
    - Market for smart devices was **not ready**
    - No major adoption initially

- Later, the team shifted focus to:
    - **Internet and web applications**

---

## 4. Transformation to Java (1995)

- Oak was renamed to:
  Java

### Why "Java"?

- Named after **Java coffee**
- Chosen as a unique and catchy name

### Major Breakthrough

- Officially launched in **1995**
- Introduced the revolutionary concept:
  Write Once, Run Anywhere (WORA)

---

## 5. Key Idea Behind Java

### Problem Before Java

- Programs were **platform-dependent**
    - Windows program → runs only on Windows
    - Linux program → runs only on Linux

### Java Solution
Java Code → Bytecode → JVM → Any System

- Java introduced:
    - **Bytecode**
    - **Java Virtual Machine (JVM)**

➡This made Java **platform-independent**

---

## 6. Role of the Internet

- In the mid-1990s, the **internet was growing rapidly**
- Java became popular because:
    - It could run inside **web browsers**
    - Supported **interactive web applications**

### Applets

- Small Java programs called **applets** were used in browsers
- Helped Java gain early popularity

---

## 7. Java's Philosophy

Java was designed with the following principles:

- **Simple** → Easy to learn
- **Secure** → No direct memory access
- **Portable** → Runs everywhere
- **Robust** → Strong error handling
- **Object-Oriented** → Real-world modeling

---

## 8. Growth of Java

- Java quickly became popular in:
    - Web development
    - Enterprise applications
    - Mobile applications (Android)

- After Sun Microsystems:
    - Java was acquired by **Oracle Corporation (2010)**

---

## 9. Why Java Became Successful

- Platform independence (WORA)
- Strong community support
- Secure execution environment
- Rich standard libraries
- Continuous evolution

---

## 10. Summary
Green Project → Oak → Java → Web → Enterprise → Modern Applications

- Started as an **embedded system language**
- Evolved into one of the **most powerful programming languages**

---

## Conclusion

Java’s journey shows how a language designed for **small devices** transformed into a **global standard for software development**.

> "Write Once, Run Anywhere" remains the core strength of Java.
