# ☕ Java Architecture

---

## Introduction

Java Architecture explains how a Java program is **compiled, converted into bytecode, and executed** on different platforms using the **Java Virtual Machine (JVM)**.

---

## 2. Components of Java Architecture

### 2.1 JDK (Java Development Kit)

- Full package for Java development
- Contains:
    - JRE
    - Development tools (javac, javadoc, debugger)


JDK = JRE + Development Tools


---

### 2.2 JRE (Java Runtime Environment)

- Provides environment to run Java programs
- Contains:
    - JVM
    - Core libraries


JRE = JVM + Libraries


---

### 2.3 JVM (Java Virtual Machine)

- Core component of Java
- Executes bytecode
- Makes Java platform-independent

---

## 4. Class Loader Subsystem

- Loads `.class` files into JVM

### Types:
- Bootstrap ClassLoader
- Extension ClassLoader
- Application ClassLoader


Loading → Linking → Initialization


---

## 5. Runtime Data Areas

### 🔹 Heap
- Stores objects
- Shared among threads

### 🔹 Stack
- Stores method calls
- Each thread has its own stack

### 🔹 Method Area
- Stores class metadata, static variables

### 🔹 PC Register
- Holds current instruction address

### 🔹 Native Method Stack
- Used for native (C/C++) methods

---

## 6. Execution Engine

- Executes bytecode

### Components:
- **Interpreter**
    - Executes line-by-line

- **JIT Compiler**
    - Converts bytecode into native code
    - Improves performance

---

## 7. Garbage Collector

- Automatically removes unused objects
- Prevents memory leaks


Used Objects → Keep
Unused Objects → Delete


---

## 8. Java Native Interface (JNI)

- Allows Java to interact with:
    - C / C++ libraries

---

## 9. Java Libraries (API)

- Predefined classes and packages
- Example:
    - java.lang
    - java.util
    - java.io

---

## 10. Working of Java Architecture

Write Program (.java)
Compile using javac
Convert into Bytecode
JVM loads bytecode
Execution by Interpreter/JIT
Output generated

---

## 11. Platform Independence (WORA)


Same Bytecode → Different JVM → Different OS


- Java runs on:
    - Windows
    - Linux
    - Mac
