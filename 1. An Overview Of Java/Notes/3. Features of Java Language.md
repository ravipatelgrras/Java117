# ☕ Features of Java

---

## Introduction

Java is a **high-level, object-oriented, platform-independent programming language** known for its simplicity, security, and robustness.

---

## 1. Simple

- Easy to learn, use, and understand
- Removes complex features of C++
    - No pointers
    - No multiple inheritance (class-based)
- Clean and readable syntax

---

## 2. Object-Oriented
Class → Object → Behavior

Encapsulation | Inheritance | Polymorphism | Abstraction

- Everything is an object
- Promotes reusability

---

## 3. Robust
Try → Catch → Finally

- Strong memory management
- Exception handling
- Type checking

---

## 4. Multithreaded
Thread 1 → Task A
Thread 2 → Task B
Thread 3 → Task C

- Executes multiple tasks simultaneously
- Better CPU utilization

---

## 5. Architecture-Neutral
Source Code → Bytecode → Any Machine

- No hardware dependency

---

## 6. Interpreted

- Bytecode executed by JVM
- Platform independent
- Easy debugging

---

## 7. High Performance
Bytecode → JIT → Native Code → Fast Execution

- Uses JIT compiler
- Faster execution

---

## 8. Distributed

- Supports TCP/IP, RMI
- Used for network applications

---

## 9. Dynamic
Class Loading → Linking → Execution

- Runtime class loading
- Reflection support

---

## 10. Platform-Independent
Java Program → Bytecode → Windows | Linux | Mac

- WORA concept

---

## 11. Secure

- No direct memory access
- Bytecode verifier
- ClassLoader security

---

## 12. Portable

- No platform-specific features
- Easy to move across systems

---
