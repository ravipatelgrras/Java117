# 📘 Java Software Interview Questions (Q/A)

---

## 1. What are the Types of Java Editions?

**Answer:**
Java provides different editions for different types of applications:

### 🔹 Java SE (Standard Edition)
- Core and foundation of Java
- Provides basic APIs and features
- Used for desktop and standalone applications

---

### 🔹 Java EE (Enterprise Edition) / Jakarta EE
- Built on top of Java SE
- Used for large-scale enterprise applications
- Provides APIs for web, distributed, and secure systems

**Examples:**
- Banking systems
- E-commerce applications

---

### 🔹 Java ME (Micro Edition)
- Lightweight version of Java
- Designed for resource-constrained devices

**Used in:**
- Embedded systems
- Smart cards
- Early mobile devices

---

### 🔹 JavaFX
- Used for building modern GUI applications
- Supports graphics, media, and animation
- Replacement for AWT and Swing (modern UI)

---

## 2. What is the difference between JDK, JRE, JVM & JIT?

**Answer:**

- **JDK:** Development kit (includes JRE + tools like compiler)
- **JRE:** Runtime environment (includes JVM + libraries)
- **JVM:** Executes bytecode and converts it into machine code
- **JIT:** Improves performance by compiling bytecode into native code at runtime

---

## 3. What is Java plug-in?

**Answer:**
- Used to run Java programs in web browsers (applets)
- Now obsolete due to security issues

---

## 4. What is Java SE & what is the difference between Java SE & JDK?

**Answer:**

- **Java SE:** Platform providing APIs
- **JDK:** Tool used to develop Java applications

---

## 5. What is the meaning of Major & Minor versions of Java?

**Answer:**
- **Major Version:** Big updates (Java 8, 11, 17)
- **Minor Version:** Small updates/patches (1.8.0_202)

---

## 6. What is the difference between Version & Stable version?

**Answer:**
- **Version:** Any release
- **Stable Version:** Tested and reliable version

---

## 7. Will JVM installed automatically with OS?

**Answer:**
❌ No  
✔ Comes with JRE/JDK

---

## 8. Do we have single Java software for all types of OS?

**Answer:**
❌ No  
✔ Different OS require different Java installations

---

## 9. What are the important folders & files of Java software?

**Answer:**

### Important folders:
- **bin/** → tools (javac, java)
- **lib/** → libraries
- **include/** → header files

### Important files:
- **javac.exe** → compiler
- **java.exe** → interpreter
- **rt.jar** → core classes

---

## 10. Why Java is called open source software?

**Answer:**
- Java is partially open source
- OpenJDK is freely available
- Source code can be accessed and modified
