# 📘 Java History Interview Questions (Q/A)

---

## 1. Why Java language is developed?

**Answer:**
Java was developed to create **platform-independent, secure, and portable applications**.

**Main reasons:**
- To overcome platform dependency of C/C++
- To support network-based applications
- To provide secure and robust programming
- Initially developed for embedded systems (TV, set-top boxes)

---

## 2. When should we use C language & when should we use Java language?

**Answer:**

### Use C when:
- System programming (OS, drivers)
- High performance required
- Direct hardware interaction

### Use Java when:
- Web, enterprise, or mobile applications
- Platform independence required
- Security and OOP features needed

---

## 6. What is a platform, Platform Dependency and Platform Independency?

**Answer:**

- **Platform:** Hardware + Operating System
- **Platform Dependent:** Runs only on specific OS (C, C++)
- **Platform Independent:** Runs on any OS (Java)

---

## 7. What is the meaning of:

### a. Source code, Compiled code, Executable code

- **Source Code:** Human-readable code (.java)
- **Compiled Code:** Bytecode (.class)
- **Executable Code:** Machine-level code

---

### b. Compilation, Execution

- **Compilation:** Source → Bytecode
- **Execution:** Bytecode → Run using JVM

---

### c. Compiler & Interpreter

- **Compiler:** Converts full code to bytecode (javac)
- **Interpreter:** Executes bytecode line by line (JVM)

---

## 8. Why C, C++ programming languages are platform dependent?

**Answer:**
- Compiled into machine-specific code
- Depends on OS and hardware
- Needs recompilation for different platforms

---

## 9. How Java achieved Platform Independency?

**Answer:**
Java uses **Bytecode + JVM**

**Process:**
1. Source code → Bytecode
2. Bytecode → JVM
3. JVM → Machine code

✔ Same bytecode runs everywhere

👉 **WORA: Write Once, Run Anywhere**

---

## 10. Why JVM platform dependent?

**Answer:**
- JVM interacts with OS
- Each OS needs its own JVM

Examples:
- Windows JVM
- Linux JVM

---

## 11. What is Java’s Slogan?

**Answer:**

👉 **Write Once, Run Anywhere (WORA)**

---

## 12. Explain why Java is called language, platform, & technology?

**Answer:**

### Java as a Language:
- Used to write programs

### Java as a Platform:
- Provides JVM + JRE

### Java as a Technology:
- Includes:
    - JVM
    - JDK
    - APIs
    - Frameworks (Spring, Hibernate)
